# Customer Journey Map

This journey map tracks our persona's experience stage-by-stage as they interact with the HDI Predictor application, identifying touchpoints, thoughts, feelings, ownership, and opportunities.

---

## 🗺 Journey Map

| Phase of Journey | Stage 1: Inputting Indicators | Stage 2: Analyzing Predictions | Stage 3: Reviewing & Exporting |
| :--- | :--- | :--- | :--- |
| **Actions** *What does the customer do?* | 1. Navigates to dashboard<br>2. Selects a reference country to prefill values<br>3. Adjusts parameters using sliders or inputs | 1. Clicks "Compute Prediction"<br>2. Observes large HDI score<br>3. Reviews progress dimensions | 1. Reads strategic policy suggestions<br>2. Clicks "Export Report PDF"<br>3. Downloads JSON data |
| **Touchpoint** *What do they interact with?* | * Web dashboard navbar<br>* Country dropdown menu<br>* Form fields and range inputs | * Gradient header banner<br>* Dimensional progress bars<br>* Model confidence indicators | * Suggestion cards (icons)<br>* Export PDF button<br>* History table |
| **Customer Thought** *What are they thinking?* | * "It is great to prefill these fields rather than looking them up."<br>* "The interface is very fast." | * "This predicted value looks highly accurate."<br>* "I can clearly see which component is lagging." | * "The suggestions correspond to our development guidelines."<br>* "I can print this page directly to PDF." |
| **Customer Feeling** | **Satisfied / Confident** | **Informed / Clear** | **Accomplished / Productive** |
| **Process Ownership** *Who is in the lead?* | UI Engineer (Input components stability) | ML Engineer (Model accuracy, scaling) | Backend & QA Engineers (Export triggers, recommendations logic) |
| **Opportunities** *How can we improve?* | Add a search-filter to the country dropdown for faster lookup. | Add comparison charts comparing the predicted country against regional peers. | Support direct integration with Google Drive or Dropbox for cloud report saving. |
